# GitHub User Finder

## Автор: [Никита Чернецов]
## Описание:
Приложение для поиска пользователей GitHub с возможностью добавления их в избранное и сохранения списка в файл JSON.
Использует GitHub API, Tkinter для GUI и Git для контроля версий.

## Технологии:
- Python 3.x.
- Tkinter (GUI).
- Requests (GitHub API).
- JSON (сохранение избранного).
- Git.

## Как использовать API:
1. Введите логин пользователя GitHub в поле поиска.
2. Нажмите «Поиск».
3. Выберите пользователя из списка и нажмите «Добавить в избранное».
4. Избранные пользователи сохраняются в файл `favorites.json`.
5. Для просмотра избранных нажмите «Мои избранные».

## Примеры использования:
- Поиск существующего пользователя: введите `torvalds`.
- Поиск несуществующего пользователя: введите `nonexistentuser123` (появится сообщение о том, что пользователь не найден).
- Добавьте пользователя в избранное и проверьте файл `favorites.json`.
